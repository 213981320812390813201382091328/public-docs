# מדיניות צוות בחינות 18+

הקהילה מיועדת לחברי קהילה המלאו להם 18+

# פתיח
צוות מתנדבים יקר, תודה שאתם לוקחים חלק ועוזרים ותורמים לקהילה שלנו!
